# Dossier des images visibles dans `resources#gallery`

Placez ici les images qui doivent s'afficher dans la galerie de la page `Resources`.

Structure attendue :

- `public/resources/gallery/cardano-summit-2022/photo-1.jpg` ... `photo-6.jpg`
- `public/resources/gallery/cardano-africa-tech-summit/photo-1.jpg` ... `photo-6.jpg`

Remarques :

- Les noms de fichiers suivent le mapping utilisé dans `src/pages/Resources.tsx`.
- Vous pouvez remplacer les images sans modifier le code si vous gardez les mêmes noms.
- Si Strapi fournit des images pour `gallery-events`, ces images CMS prendront la priorité.
